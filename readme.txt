1️⃣ What is the difference between var, let, and const?
 Answer:
     var:It's a old way to declare variables. WE Can redeclare and update.

     let:It's a modern way to declare variables. WE Can update but not redeclare.

     const:We use it when the value will not redeclare or update and using con we Cannot redeclare or update.
